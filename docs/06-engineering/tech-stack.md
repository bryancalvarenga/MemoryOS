# Technology Stack

## Frontend

Framework: React-based framework

Responsibilities:

- user interface
- document management
- search interface

## Backend

Runtime: Node.js

Framework: structured backend framework

Responsibilities:

- API endpoints
- business logic
- authentication
- orchestration

## Database

Relational database:

PostgreSQL

Used for:

- user data
- document metadata
- query logs

## Vector Search

Vector extension:

pgvector

Used for:

- semantic search
- similarity queries

## File Storage

Object storage compatible with S3.

Used for storing uploaded documents.

## Processing

Background jobs handle:

- text extraction
- chunk generation
- embedding generation
