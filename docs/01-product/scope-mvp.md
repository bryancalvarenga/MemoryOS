# MVP Scope

## Purpose

This document defines the boundaries of the initial version of MemoryOS.

The objective is to build a focused system that validates the core value of semantic memory retrieval.

## Features Included

The MVP will include the following features:

### Authentication

- user registration
- login
- session management

### Document Ingestion

Users can upload:

- PDFs
- markdown files
- bookmarks

### Processing Pipeline

Documents will go through:

1. text extraction
2. chunk generation
3. embedding generation
4. vector indexing

### Search

Users can perform semantic search queries and retrieve relevant document fragments.

### Content Management

Users can:

- view imported documents
- delete stored content

## Features Excluded

The following features are intentionally excluded from the MVP:

- AI recommendations
- automated knowledge connections
- team collaboration
- integrations with multiple external platforms
- knowledge timeline reconstruction
- automated insights
- memory prediction systems

These features may be considered for future versions.

## MVP Success Definition

The MVP succeeds if users can reliably retrieve previously imported knowledge using semantic search.
